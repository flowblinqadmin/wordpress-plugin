AI visibility optimization for your WordPress site — powered by Flowblinq GEO.

== Description ==

Flowblinq GEO audits your site for AI engine visibility, generates an llms.txt file, and automatically injects structured data to help AI models understand and cite your content.

Features:
* One-click GEO audit submission
* Automatic llms.txt generation and serving
* Schema.org JSON-LD injection
* Before/after optimization scoring

Requires a free Flowblinq account and API credentials. Get started at https://geo.flowblinq.com.

== Installation ==

1. Upload the `flowblinq-geo` folder to `/wp-content/plugins/`.
2. Activate the plugin in WP Admin → Plugins.
3. Go to Settings → Flowblinq GEO and enter your Client ID and Client Secret.
4. Click "Run Audit" to start your first GEO audit.

== Frequently Asked Questions ==

= Where do I get my API credentials? =

Sign up at https://geo.flowblinq.com, then go to Settings → API to generate a Client ID and Client Secret.

= How long does an audit take? =

Typically 2–5 minutes depending on site size.
